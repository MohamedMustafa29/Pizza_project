 Pizza_project
A desktop application built using C# and Windows Forms. It simulates a pizza ordering process where users can customize their order and see the total price in real-time.

✨ Features
Size Selection: Choose between Small, Medium, and Large.

Toppings: Add various toppings like Extra Cheese, Mushrooms, Tomatoes, etc.

Crust Customization: Toggle between Thin and Thick crust.

Dining Options: Select "Eat In" or "Take Out".

Dynamic Order Summary: Real-time updates for the selected options.

Total Price Calculation: Automatically calculates the final price based on selections.

Reset Functionality: Clear the entire form with a single click.
